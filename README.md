# FastAPI-Pokedex

```shell
# CMD with Uvicorn ASGI or FastaAPI-CLI

# Development mode
fastapi dev --port <Your Port (Default: 8000)>

# Production mode
uvicorn main:app --host 0.0.0.0 --port 9090 --workers 4;
fastapi run --host 0.0.0.0 --port 9090 --workers 4;
```
